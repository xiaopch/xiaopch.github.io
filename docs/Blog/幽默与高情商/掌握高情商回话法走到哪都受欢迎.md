# 掌握高情商回话法走到哪都受欢迎

  任何人跟你说，谢谢你都不要说不用谢，也不要说不客气。你要是这么回复，会让人觉得你帮了好像没帮似的，什么也没得到，会说话，不如会接话，会接话的人，不管会到哪里，那都是人见人爱。做什么事儿，别人都愿意帮你。接下来内容，一定要看到最后。套用这些公式和话术，我保证你说完之后，别人对你瞬间好感度拉满。

- 去便利店买东西，一个人说你的名字好好听啊，不要说谢谢夸奖，你的名字也很好听，要说， **你念出来好像更好听了**。

- 新来的同事跟你说：我不太会说话，你不要说没有。你要说：**没关系，你慢慢说，我慢慢听。**

- 上班总掐点打卡，领导说，你挺会卡点的嘛？高情商回答，**如果连时间都掌控不了，怎么会掌控人生？**

- 同事见到说，你好聪明啊，你就回答，**近朱者赤，和你在一起，熏都熏出来了。**

- 男朋友夸你，你的衣服好好看呀，高情商回答，谢谢。**因为今天要见你，所以穿的好看一点。**

- 领导夸你进步大，不要说，没有没有。要回复，**谢谢领导，强将手下无弱兵，我这必须进步，快点跟上队伍的节奏啊，要不然都不敢说是您手下的兵。**

- 强烈推荐。大家看看这本《回话有分寸》。
