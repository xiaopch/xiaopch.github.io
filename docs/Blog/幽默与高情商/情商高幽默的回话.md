# 老爸当我的恋爱嘴替

- 女生：你竟然有双休，你的生活，我的梦啊。

- 男生：那你咋还不追梦？



- 女生：我年纪大了，追不动了。那我可以追你吗？

- 聊而不追，才能进可攻退可守，让她以为关系是她推进的。其实把握节奏点是你。

- 男生可以这么回答：你说你尽管追吧，我腿脚不好，跑不了多远。
  
  

- 女生：今天下班回到家里空荡荡的，突然感觉挺失落的。

- 男生：你家可能缺个男主人。

- 女生：我以前不这样的，其实我是个挺开朗的人。

- 男生：我就叫挺开朗。
  
  

- 女生：你好搞笑，我差点没反应过来，跟你聊天还挺开心的。

- 男生再幽默的试探一下：你家男主人的岗位招人吗？明天我想约个时间面试一下。



- 女生：招人，那就明天中午吧，你可别忘了带简历。

- 男生：我们不见不散。

- 怎么样？学去吧。
