# 全能提示词手册

## 核心万能公式

**设定角色 + 明确任务 + 细节要求 + 输出格式 + 风格语气 + 字数篇幅 + 禁止雷区**
所有需求套这个框架，AI 输出直接精准不跑偏。

# 一、写作作文专属模板

### 1. 初中生记叙文

你是资深初中语文优生作文辅导老师，按**题记+三个小标题**结构，写一篇【____】主题记叙文，字数600-800字，文笔细腻有文采，有细节描写、首尾呼应，立意积极向上，无网络口水话，贴合初二水平。

### 2. 议论文

帮我写一篇【____】主题议论文，总分总结构，有论点、2-3个分论点+事例论据，逻辑严谨，语言精炼，700字左右，适合考场作文。

### 3. 文案润色

帮我润色下面这段文字，提升文采和高级感，不改变原本原意，语句通顺流畅，去掉口语废话：
【粘贴内容】

### 4. 仿写句子/段落

仿照例句的句式、修辞和意境，仿写3句，风格保持一致：
例句：【____】

# 二、学习刷题专属模板

### 1. 知识点总结

帮我梳理【科目+章节】核心考点，分点罗列，精简干货，只留考试必背内容，条理清晰，适合考前速记。

### 2. 逐题精讲

把这道题用**分步讲解**，先给思路、再给步骤、最后标注考点+易错点，用初中生能听懂的大白话，不跳步骤：
【发题目】

### 3. 出练习题

按初二【____】知识点，出10道基础+拔高题，选择填空+简答搭配，难度适中，题目后附标准答案，排版整齐。

### 4. 英语专项

帮我翻译这段英文，逐句直译+通顺中文意译；再标出重点短语和语法考点：
【粘贴英文】

# 六、人设对话&情绪疏导模板

### 1. 心理疏导模式

你现在扮演专业心理疗愈师，语气温柔低沉、语速放缓，耐心倾听我的情绪，共情安慰，不评判、不说教，慢慢引导我舒缓心情。

### 2. 学霸陪聊模式

你是温柔耐心的初二学霸同学，我有学习和生活问题都可以问你，讲解通俗易懂，态度亲和，不敷衍。

# 七、通用万能指令（随手套用）

1. 用大白话重新解释一遍，通俗易懂，零基础也能看懂
2. 把内容分点整理，条理清晰，不要大段文字
3. 缩短一半字数，保留核心，精炼简洁
4. 换成正式书面语，适合作文/作业使用
5. 帮我排查这段话的语病，修改通顺并优化表达
