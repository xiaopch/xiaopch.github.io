# 《安心正念课》主要内容总结

**作者**：赵安安（安安老师）  
**出版社**：机械工业出版社，2018年  
**页数**：195页

---

## 📖 全书概览

本书以**正念**为核心理念，以**提高人生的掌控力**为最终目标，采用**8周课程**的结构，由内而外、逐步升华，带领读者从呼吸练习开始，最终成为一个身心完整的自由之人。全书融合了东方禅修智慧与西方科学验证，既有理论深度，又有实操练习。

---

## 🧘 核心概念

**正念 = 带着慈悲和放松，专注在每一刻的当下。** 它是一种不评判的、随时随地的觉察，不只觉察外在的人事物，更要觉察自己。作者强调，正念不是宗教修行，而是经过科学验证的、系统化的心理训练方法。

---

## 📑 十章内容结构

### 第一部分：基础理念（第1-2章）

**第一章 正念是什么**

- 正念是**当下的专注与觉察**
- 五大益处：
  - 🟢 **提高快乐水平**：练习8周后大脑神经回路改变，左前额叶更活跃，积极情绪增多
  - 🟢 **增加慈悲心与同理心**：前额叶和脑岛增厚，"社交脑"功能增强
  - 🟢 **提高工作效率，降低压力**：大脑白质效率提高，决策更理性
  - 🟢 **促进创造力**：开放式觉察带来"不设限"和"重新组合"两大创造力要素

**第二章 正念练习的9种态度**

| 态度    | 核心要义                 |
| ----- | -------------------- |
| 初心    | 每次体验都像第一次，保持赤子之心     |
| 接纳    | 不纠结的秘诀，接纳现状才能站稳脚跟    |
| 信任    | 自我实现的预言，信任自己才能改变命运   |
| 耐心    | 快与慢并非绝对，生命需要时间酝酿     |
| 放下    | 放不下就先放着，给心灵留出空间      |
| 感谢    | 反馈生命中的美好             |
| 施予    | 与世界积极联结              |
| 不评断   | 看见更多可能性的智慧，有判断力但不生好恶 |
| 不用力追求 | 不费力就达成目标             |

### 第二部分：8周实操课程（第3-10章）

**第3章·第一周：正念从呼吸开始**

- 核心：留意呼吸，觉察自己，将溃散的注意力一次次收回
- 练习：**观呼吸** + **正念进食**
- 关键：专心走路、专心吃饭，活在当下

**第4章·第二周：把专注带进身体**

- 核心：认识练习中的5种障碍（贪欲、恶意、昏沉、掉举、疑），四步摆脱
- 练习：**身体扫描** + **正念行走**
- 关键：将注意力如探照灯般照向身体各部位

**第5章·第三周：让身体与呼吸合二为一**

- 核心：身心合一是"全人"状态，学习"默照"功夫（止观双运）
- 练习：**身心合一的呼吸** + **正念伸展**
- 关键：身体在这里，心也在这里

**第6章·第四周：解码情绪的反应**

- 核心：觉察大脑的"自动化导航系统"，写心情日记
- 练习：**观情绪**
- 关键：让情绪来去不执着，练习正念认知疗法

**第7章·第五周：看见头脑里的念头**

- 核心：识别常见思维误区，写"正念观虫日记"，用后设认知应对
- 练习：**观声音和念头**
- 关键：心念是我们自己制造的幻境，看见"虫虫"但不被它带走

**第8章·第六周：与烦恼和平共处**

- 核心：**接纳为改变之母**，放下好恶之心，不评判自己的评判
- 练习：**与烦恼和平共处** + **正念体操**（葛吉夫律动）
- 关键：你是天空，烦恼只是云；穿越乌云，背后是幸福的光线

**第9章·第七周：处理人际关系**

- 核心：正念沟通三步骤（觉察自己→倾听对方→表达自己）
- 练习：**慈悲心的练习** + **内在小孩的练习** + **正念沟通日记**
- 关键：用"绿色沟通"取代"红色沟通"，从爱自己到爱他人

**第10章·第八周：成为一个完整的人**

- 核心：整合所学，让正念成为生活方式，找到人生的"calling"（天职）
- 练习：综合所有练习，形成日常习惯
- 关键：一花一世界，一沙一天堂，把每件小事做好，生命道路自然展开

## 🔬 科学基础

本书大量引用了科学研究：

- **脑科学**：8周正念练习后杏仁核缩小、前额叶增厚；长期练习者脑结构改变
- **心理学**：正念认知疗法（MBCT）、自我实现预言、损失厌恶等经典理论
- **医学**：正念对心血管疾病、慢性疼痛、焦虑抑郁等有改善作用
- **苹果iOS**将正念列入健康四大支柱（与健身、营养、睡眠并列）

## 📝 总结

这是一本**入门友好、结构清晰、知行合一**的正念实践指南。它最独特之处在于：既有东方禅修的文化底蕴，又有西方科学的实证支撑；既讲"为什么"，更重"怎么做"。每周配有音频/视频引导练习、作业和问答环节，读者可以像上课一样循序渐进地完成8周正念之旅，最终实现从"知道"到"做到"的蜕变。
