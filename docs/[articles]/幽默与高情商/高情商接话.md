# 高情商接话

- 1.当别人对你说：谢谢你。可以回：嗨，**别这么见外，以后我可能还得找你帮忙呢**。

- 2.当别人夸你：性格好！你就回:“**那也要看是对谁**，你对我那么好，我当然要温柔啊，其他人就没这待遇了。”

- 3.当你耽误了别人时间，不要说对不起，是我的错；可换成：谢谢你愿意花时间陪我。

- 4.别人说：我不太会聊天。你就回：**不是不会聊，只是没碰到对的人，你看，我们聊得多好。**

- 5.夫妻之间，征求意见，不要回复：随便。可以这样回：**你做主吧，我喜欢听你的意见。**

- 6.对别人讲完一件事情，不要问：你听懂没有；要问：**我讲明白了吗。**

- 7.别人说你是宅女。幽默高情商回复：**不要叫我宅女，叫我居里夫人**。

- 8.别人问: 电话怎么一直打不通?你就回：**最近在谈一个100块的大生意，资金不够，欠费了。**

- 9.别人问：你喜欢什么类型的？你可以回：**我喜欢的就你这个类型的**！

# 高情商小技巧

---

| 场景           | 低情商回复 (❌)             | 高情商回复 (✅)                                       |
| ------------ | --------------------- | ----------------------------------------------- |
| 领导问：在不在      | 在的 / 领导您找我有什么事 / 怎么了？ | 在的，领导您有什么安排吗？                                   |
| 领导说：谢谢       | 不用谢。                  | 领导您客气了，您先忙，有问题随时找我。                             |
| 领导说：这两天工作辛苦了 | 不辛苦。                  | 谢谢领导关心，这两天接触到了不少新工作，虽然很累，但是学到很多东西，有不足的地方还请您多指教。 |
| 领导问：现在有空吗？   | 没空。                   | 领导，有新工作安排吗？如果急的话，我先把手头的工作放一放，您请说。               |
| 领导安排你完成不了的工作 | 领导，我干不了。              | 好的，领导，不过我缺乏经验，有些事情还需要您的帮助和指导。                   |
| 和领导请示工作      | 领导跟我说一下这事怎么办。         | 领导，有件事情需要请示您。                                   |
| 担心领导没听明白     | 您听明白了吗？               | 领导，以上我都说清楚了吗？                                   |
| 犯错被问怎么回事     | 我也没想到事情会这样。           | 领导，还是我年轻，缺乏经验，当时考虑得不够全面，我的补救措施，第一、第二、第三……       |
| 领导问：工作进展怎样了？ | 正在做。                  | 具体地说，哪部分已经做完了，哪部分还在进行到哪个阶段，预计本周内能够全部完成。         |
| 领导问你不知道的事    | 我不知道。                 | 领导，我详细了解一下，然后立刻向您汇报。                            |
