# 回话高手-记住这10条回话技巧

给大家分享几个高情商的接话法。对于你提升人际关系非常的有用。

- 第一、面对问候的时候，不要说还行，要换成**挺好的呀，谢谢关心！你呢，最近一切都还顺利吧，好的交互一定是有来有往。**

- 第二、面对批评的时候，要淡定明智的对待他人的批评。可以说您说的对，这确实是我需要改进的地方，**感谢您的指正，我会努力改正的。**

- 第三、面对朋友烦恼的时候，不要说别想那么多，要说**我相信你能够克服这个困难，咱们一起想办法解决，我相信你一定能跨过去的。**

- 第四、面对夸奖的时候，不要说哪里哪里，过奖过奖。**而是要换成非常感谢你的认可，其实只做了自己该做的，能得到您的肯定，我真的很开心。**

- 第五、碰到厉害的人，别说您真牛，要说**原来高手就在我身边呀**。

- 第六、对陌生人，不要问叫什么名字，要说**我应该怎么称呼您**。

- 第七、夫妻之间，不要经常说，随便都可以，**而要换成我喜欢听你的**。

- 第八、饭局上，别人夸你酒量好，不要说一般一般。**而要说酒量好不好，那得看跟谁喝。今天跟您喝心情好，就多贪了两杯。**

- 第九，当客人说菜店多了时，不要说不多不多，而要说你好不容易来一次，当然得好好招待下，也不知道这些菜合不合您的口味，您尝尝看，不合适，咱再换。有招待不周的地方，请您多包涵。

- 第十，有人给你发红包，你不想收，**你就回心意和钱，我就捡更贵重的收下，红包我就不领了，谢谢。**

- 第十一、生活再艰难，也不要说过不下去了，烦死了，累死了，而要换成**不破不立，人生已经走到谷底了，往哪走，走都是向上。**

- 希望这些对你有所启发。
