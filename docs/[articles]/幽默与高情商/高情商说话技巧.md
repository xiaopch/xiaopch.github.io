# 提高情商的这三个技巧

 让别人一提到你，就都认为你情商高，感觉和你相处很舒服，而且还想继续和你深入的交往。

1、聊天的时候，要多引导对方去谈他自己的事儿。

研究表明，一段让人舒服的谈话，百分之八十是聊对方的，百分之二十才是聊自己。

那怎么引导对方呢？三个字，多提问。

 比如说：后来呢？然后呢？这戒指你在哪买的？那哥们儿后来怎么样了？这个技巧用的是镁光灯效应，就是对方感觉到你对他的事儿感兴趣，那他就会喜欢你。

2、记住对方的小细节

比如a喜欢吃香菇，b不吃葱，你想一一下大伙一起出去吃饭，有个人就说哎，这道菜别放葱，他不吃葱。如果啊不吃葱的这个人是你，你对他会是什么感觉？被重视，被挂念是每个人的底层需求。 谁给他这个，他就对谁有好感。

3、把对话里面的你全都换成我，或者是我们咱们

这是心理咨询师的基本操作，为的是跟咨询者迅速拉近关系。

比如你原来要说的是，这事儿，你可以这么想。你把这句话给他换成，这事儿，咱们可以这么想。是不是感觉就不一样了？你这个字儿，会营造一种对立的姿态。

我们，咱们这样的词儿，可以构成共同体的思维，让别人觉得你们是一伙的。
