Seafile私有云使用说明

发表于 2018-01-15

Seafile的中文名叫海文网盘，致力于安全、可靠、快速的文件同步和共享，有开源版和企业版两个版本。开源版已经可以满足个人和小团队的文件同步和共享需求。Seafile代码开源，也提供了Raspberry Pi版本的服务器软件。
根据官方报告，截至 2016 年初，Seafile 已经有超过 30 万用户使用，并拥有卡巴斯基，德国 Mainz 大学，德国 HU Berlin 大学和法国 Strasbourg 大学等大型组织机构客户。

项目首页：https://www.seafile.com

下载页：https://www.seafile.com/download/

下载地址：https://github.com/haiwen/

下载得到seafile-server_x_stable_pi.tar.gz文件。

安装Seafile到Raspberry Pi
seafile使用Python开发，先安装依赖包：

sudo apt-get install python2.7 python-setuptools python-imaging python-ldap sqlite3 python-urllib3

将seafile-server_6.0.5_stable_pi.tar.gz上传到Raspberry Pi，并解压缩。

tar -xvf seafile-server_6.0.5_stable_pi.tar.gz
cd seafile-server-6.0.5

按ENTER继续，根据提示依次输入服务器名字，服务器IP/域名，数据目录，端口号。

开始运行安装程序：

提示seafile fileserver端口是8082，seahub端口是8000。
启动seafile：

./seafile.sh start

按ENTER，接下来设置seahub
启动seahub：
./seahub.sh start 8000

首次启动提示建立管理员账号。

按提示依次输入管理员邮箱，密码，第二遍密码完成管理员账号的创建。
在浏览器中打开：
http://192.168.2.169:8000

（192.168.2.169是Raspberry Pi的IP，请根据实际情况修改），Seafile导航到了登录页，使用刚才创建的管理员账号登录，稍等片刻，显示欢迎界面
