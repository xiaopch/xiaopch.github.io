todolist开源和git使用方法

发表于 2018-02-24

todolist简化版本使用方法

　　自己在家里树莓派上用的blog，基于python的flask框架。没有用户管理，通过session+get登陆。
通过uid进行性分类。

修改源自：https://github.com/lalor/
项目地址：https://github.com/xiaopch/todolist

运行blog服务

python app.py

进入blog，输入网址：

http://127.0.0.1/?t=xp&uid=1
#t为session密码
#uid为任意较小的整数，1，2，3,......

download文件夹为下载目录

github上传

运行git bash，开始配置

git config --global user.name xiaopeng
git config --global user.email user@163.com
ssh-keygen -C 'user@163.com' -t rsa

  • 如果一切顺利的话，可以在用户主目录里找到 .ssh 目录，里面有 id_rsa 和 id_rsa.pub 两个文件，这两个就是SSH Key 的秘钥对，id_rsa 是私钥，不能泄露出去，id_rsa.pub是公钥，可以放心地告诉任何人。用记事本打开 id_rsa.pub（C:\Users\Administrator.ssh），得到ssh key 公钥。

  • 为 Github 账户添加 ssh key 。登录 Github，展开个人头像的小三角，点settings，然后打开SSH keys菜单，点击Add SSH key新增密钥，填上标题和公钥。

git bash使用方法

::获取github源程序并建立连接
::git clone https://github.com/xiaopch/share.git
::建立github连接
::git remote add origin git@github.com:xiaopch/share.git

::显示连接的github repository
git remote -v  
git add .
::将本地文件或文件夹同步到本地缓存
git commit -m 'cmd'
git push origin master

git gui配置

账号保存

　　若不做设置的话，每次提交的时候，都会询问你填写密码。于是我们先来把这个设置好。

添加环境变量

　　我的电脑 - 属性 - 高级系统设置 - 环境变量 - 新建变量

　　变量名HOME，变量值%USERPROFILE%

创建账号文件

　　开始 - 运行 中打开%Home%，即windows的管理员账号文件夹。新建一个名为“_netrc”的文件，填写你要保存的服务器地址及账号密码，保存。 

machine github.com
login ******@***.com
password ******
