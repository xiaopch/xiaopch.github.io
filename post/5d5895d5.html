MongoDB 与 RDBMS Where 语句比较

操作 格式 范例 RDBMS中的类似语句
等于 {:} db.coll.find({"by":"教程"}).pretty() where by = '教程'
小于 {:{$lt:}} db.coll.find({"likes":{$lt:50}}).pretty() where likes < 50
小于或等于 {:{$lte:}} db.coll.find({"likes":{$lte:50}}).pretty() where likes <= 50
大于 {:{$gt:}} db.coll.find({"likes":{$gt:50}}).pretty() where likes > 50
大于或等于 {:{$gte:}} db.coll.find({"likes":{$gte:50}}).pretty() where likes >= 50
不等于 {:{$ne:}} db.coll.find({"likes":{$ne:50}}).pretty() where likes != 50
