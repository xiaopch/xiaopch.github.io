github配置和上传

发表于 2018-07-18

生成SSH Key

  • Git Bash:
    ssh-keygen -t rsa -b 4096 -C xiaopch@163.com
  • 不需要输入,保持密码为空

为github账号配置ssh key

cat ~/.ssh/id_rsa.pub
  • 切换到github，点击settings，然后打开SSH keys菜单，点击Add SSH key新增密钥，填上标题（最好跟本地仓库保持一致）。

上传mydir目录的文件到github

cd mydir
git init
git add .
git commit -m add new
::git remote rm origin
git remote add origin git@github.com:xiaopch/data.git
git push -u origin master -f

常用命令

git add -A mydir
git pull origin master
git pull --rebase origin master
git config --global user.name xiaopch
git config --global user.email xiaopch@163.com
git clone https://github.com/xiaopch/share.git
